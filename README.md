# Radixir

An Elixir wrapper for Radix APIs and functionality.

## Installation

The package can be installed by adding `radixir` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:radixir, "~> 0.0.1"}
  ]
end
```

Docs can be found at [https://hexdocs.pm/radixir](https://hexdocs.pm/radixir).
